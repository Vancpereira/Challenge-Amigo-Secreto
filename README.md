Challenge - Amigo Secreto

Sobre o Projeto

Challenge Amigo Secreto é um projeto desenvolvido para auxiliar na organização de um sorteio de amigo secreto. Ele permite adicionar nomes de participantes e realizar um sorteio aleatório entre eles.

Funcionalidades

Adicionar nomes de amigos a uma lista

Exibir a lista de participantes

Realizar o sorteio de um amigo secreto de forma aleatória

Tecnologias Utilizadas

HTML

CSS

JavaScript
